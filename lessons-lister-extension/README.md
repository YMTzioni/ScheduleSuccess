# Lessons Lister Extension

תוסף Chrome (Manifest V3) שסורק קורסים באתר Success ומייצר רשימת שיעורים מסוננת.

## התקנה מקומית

1. פתח Chrome וגש ל־`chrome://extensions`
2. הפעל מצב מפתח (Developer mode)
3. לחץ `Load unpacked`
4. בחר את התיקייה: `lessons-lister-extension`

## שימוש

1. היכנס לעמוד הקורסים באתר.
2. לחץ על אייקון התוסף.
3. לחץ `סרוק עכשיו`.
4. בתום הסריקה תופיע רשימת שיעורים בחלונית התוסף.

## סינון מובנה

המערכת מסננת אוטומטית פריטים שכותרתם מכילה:

- `מבחן`
- `משוב על הקורס`
- `מצגות`
- וגם מקבילות באנגלית (`survey`, `quiz`, `test`, `presentation`)
